# Examples

This page provides links to example scripts demonstrating various Minapari features.

## Basic Examples

### [basic_usage.py](../examples/basic_usage.py)
Simple viewer with random data and multi-channel images.

### [load_image.py](../examples/load_image.py)
Load images from files using various libraries.

### [3d_volume.py](../examples/3d_volume.py)
Display and interact with 3D volumetric data.

## Embedding Examples

### [dockable_viewer.py](../examples/dockable_viewer.py)
Embed Minapari as a dockable widget in a Qt application.

### [multiple_viewers.py](../examples/multiple_viewers.py)
Create multiple synchronized viewers.

### [custom_controls.py](../examples/custom_controls.py)
Add custom control panels alongside the viewer.

## Advanced Examples

### [live_update.py](../examples/live_update.py)
Update image data in real-time.

### [microscopy_viewer.py](../examples/microscopy_viewer.py)
Complete microscopy image viewer application.

---

## Running Examples

```bash
cd minapari/examples
python basic_usage.py
```

Make sure you have Minapari installed:
```bash
pip install -e ".[pyqt5]"
```
